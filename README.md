# Dune Analytics (dune-analytics)

Dune is a SQL-based crowdsourced blockchain analytics platform. The Dune API exposes Query Execution endpoints, pre-built Trends endpoints, Echo (real-time multichain wallet data via Sim), a Trino warehouse connector, and dbt integration. Pricing is credit-based.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/dune-analytics/refs/heads/main/apis.yml)
